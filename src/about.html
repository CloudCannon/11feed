Welcome to 11feed!

11feed is an open-source, self-hosted, RSS reader built with the static site generator 11ty. 11feed is built on a foundation of core principles:

  • No ads, upsells, or feature bloat
  • Encourage the culture of blogging, sharing, and discovery through RSS.
  • See what others are reading and bring that content into your RSS reader.
  • Generate a purely static site that can be hosted publicly on any web server.
  • Keep features, dependencies, and the implementation to the bare essentials.
  • Provide a solid base for others to build on and customize for their own needs.
  • Emphasis on security.

Features

11feed intentionally doesn't do a lot. It tries to be a really good at the things it does do.

  • Easy Import - Import your OPML file from another RSS Reader to get set up easily.
  • Easy Export - Export your feeds as an OPML file for complete data portability.
  • Dark mode - Perfect for those late-night browsing sessions.
  • Secure - All data is rigorously sanitized. Content is securely displayed within a sandboxed iframe.
  • Categories - Keep your feeds organized with categories.
  • Search - Find the posts your looking for with full search baked in powered by Pagefind
